# Welcome to Physical AI & Humanoid Robotics

This is the introduction page for the Physical AI & Humanoid Robotics textbook. This textbook enables self-learners to build and command simulated autonomous humanoid robots using ROS 2, Gazebo, NVIDIA Isaac, and Vision-Language-Action (VLA) pipelines.
